## How to run the app ?

- npm install
- npx react-native link
- npx react-native run-android

## Set up .env file

Open .env file on your favorite code editor, and copy paste this code below :

```
REACT_APP_URL_BE=localhost
REACT_APP_URL_CLOUDINARY=cloudinary
```
